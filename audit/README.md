# audit/

This directory contains independent_recheck.py, a full reimplementation of the verification: irreducibility and order checks, the sparse certificate divisions, the single-factor exhaustive window censuses, and the complete 16,777,215-window product census. It was written by Anthropic's Claude (Fable) on 21 July 2026 during the audit round, reusing no source code from the rest of this repository, with state packing, window bit order, and orbit numbering chosen independently wherever the definitions leave freedom, which is why its reported collision coordinates differ from the manuscript's while every aggregate count matches. It was written with knowledge of the claimed witness and the expected counts, so it is a reimplementation confirmation rather than a blinded independent discovery; no blinded replication has been performed. Requires Python 3 with numpy; runtime is under a minute on ordinary hardware. Its expected output is the set of figures stated in the top-level README, and any deviation is a reportable defect.
